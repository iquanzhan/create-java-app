# create-java-app

#### 介绍
Java单模块开发脚手架，提供基础架构供项目可以直接使用。使用Spring Boot2.2.2、Mybatis Plus、Swagger等技术实现。

#### 内置功能
1.内置集成逻辑删除、分页插件
2.Swagger接口文档
3.lombok
4.hutool
5.druid
6.jsr349验证
7.mybatis-plus-generator 代码生成器
8.fastjson


#### 使用说明

1.  clone下载直接使用
